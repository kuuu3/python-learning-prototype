# Python 初學者錯誤診斷與適性練習 Prototype

這是一個純前端 MVP，針對 Python 初學者常見錯誤概念進行診斷、提示、修正、遷移題與學習摘要。

## 功能

- 初始診斷
- 主題練習：list 索引、range、條件判斷、型態轉換
- 三層提示
- 錯誤概念對應補救內容
- 自我說明關鍵概念偵測
- 總複習
- 學習紀錄與狀態轉移紀錄
- 管理頁：題庫統計、驗收檢查、展示資料、匯出 JSON
- localStorage 本機儲存

## 使用方式

直接用瀏覽器開啟 `index.html`。

## 技術

- HTML
- CSS
- JavaScript
- localStorage
